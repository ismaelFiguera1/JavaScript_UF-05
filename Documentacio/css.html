Introduccio al CSS

El CSS serveix per donar color i estructurar les pagines web. Amb HTML fem l'estructura i amb CSS ho deixem tot mes bonic.

Vam començar escrivint els estils dins de l'HTML amb l'etiqueta <style>, i despres ho vam passar a un arxiu apart que es diu style.css.

Es pot fer servir per canviar colors, marges, posicions... Els estils s'apliquen a les etiquetes com h1, div, p, etc.

Com afegir estils

Vam veure diferents maneres de posar estils CSS a una pagina web.

La primera era amb <style> al HEAD de l’HTML. Alla hi posavem els estils com colors o tamanys i funcionava directament.

Despres vam fer-ho amb un fitxer CSS apart, que es deia style.css. A l’HTML es posava un <link> al head per connectar-lo i funcionava igual pero quedava mes be.

Tambe existeix una manera que es posar l’estil en linia, dins l’etiqueta, amb style="", pero a clase no ho vam fer. Exemple: <p style="color:red">Hola</p>

Finalment, hi ha una forma mes avançada que es tenir dos fitxers CSS, i que un linki a l’altre amb @import. L'import serveix per carregar un altre arxiu CSS dins del CSS principal. Així es pot separar el CSS en diferents fitxers.
