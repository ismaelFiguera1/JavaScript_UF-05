Introduccio al CSS

El CSS serveix per donar color i estructurar les pagines web. Amb HTML fem l'estructura i amb CSS ho deixem tot mes bonic.

Vam començar escrivint els estils dins de l'HTML amb l'etiqueta <style>, i despres ho vam passar a un arxiu apart que es diu style.css.

Es pot fer servir per canviar colors, marges, posicions... Els estils s'apliquen a les etiquetes com h1, div, p, etc.

Com afegir estils

Vam veure diferents maneres de posar estils CSS a una pagina web.

La primera era amb <style> al HEAD de l’HTML. Alla hi posavem els estils com colors o tamanys i funcionava directament.

Despres vam fer-ho amb un fitxer CSS apart, que es deia style.css. A l’HTML es posava un <link> al head per connectar-lo i funcionava igual pero quedava mes be.

Tambe existeix una manera que es posar l’estil en linia, dins l’etiqueta, amb style="", pero a clase no ho vam fer. Exemple: <p style="color:red">Hola</p>

Finalment, hi ha una forma mes avançada que es tenir dos fitxers CSS, i que un linki a l’altre amb @import. L'import serveix per carregar un altre arxiu CSS dins del CSS principal. Així es pot separar el CSS en diferents fitxers.

Sintaxi CSS

Per posar estils en CSS s’ha de posar primer el selector, despres entre claus s'hi fiquen els valors

Exemple d’un estil: p { color: red; font-size: 20px; }

El selector pot ser una etiqueta com h1, una classe amb punt .titol, o un id amb hashtag #principal.

Mides

Per donar tamanys es fa servir px, %, rem i em.

px es una mida fixa en pixels. % es una mida relativa al contenidor. rem agafa el tamany segons el navegador. em agafa el tamany segons el element pare.

Exemple de mida amb pixels: font-size: 30px;
Exemple amb percentatge: width: 50%;
Exemple amb rem: font-size: 5rem;
Exemple amb em: font-size: 2em;

Tipografies

Per canviar la lletra es fa servir font-family. Es pot posar una o varies. Per exemple: Arial, Helvetica o sans-serif.

Amb font-size es pot canviar el tamany del text. Exemple: font-size: 30px;

Variables en CSS

Amb CSS es poden fer variables per guardar colors, tamanys... Les variables es fan servir posant dos guions i un nom.

Primer es declaren dins de :root i despres es fan servir amb var().

Exemple:
:root { --color-principal: red; }
h1 { color: var(--color-principal); }

Aixo serveix per canviar un color en molts llocs sense escriure'l cada vegada.

Selectors CSS bàsics

Els selectors serveixen per dir a quins elements vols aplicar els estils. Hi ha tres maneres basics: etiqueta, classe i id.

  • Etiqueta – s’aplica directament al nom de l’etiqueta. Exemple: p { color: red; }
  • Classe – es fa servir amb un punt i pots repetir-la. Exemple: .image { width: 300px }
  • Identificador – es posa amb hastag # i nomes es fa servir un cop.

Selectors CSS avançats

Hi ha selectors mes avançats que deixen fer estils mes detallats o que nomes passen en situacions concretes.

  • :hover – fa l'estil nomes quan passes el ratoli per sobre. Exemple: button:hover { background-color: red; }
  • ::first-letter – dona estil a la primera lletra. Exemple: p::first-letter { font-size: 30px; }
  • ::before i ::after – posen contingut abans o despres d’un element. Exemple: *:after, *:before { box-sizing: border-box; }
  • div > p – selecciona un parraf que esta directament dins del div. Exemple: div > p { font-style: italic; }
  • div + p – agafa el primer p que ve just despres del div. Exemple: div + p { color: green; }
  • div p – agafa TOTS els p que hi ha dintre div. Exemple: div p { font-size: 30px; }

Selectors d’atribut

A CSS es poden fer estils nomes a elements que tenen un atribut concret, com per exemple type, href o alt.

Es fa posant el nom de l’etiqueta i entre claudators l’atribut. Exemple: input[type="text"]

  • input[type="text"] – agafa els inputs de text
  • a[target="_blank"] – agafa els links que obren en pestanya nova
  • img[alt] – agafa imatges que tenen atribut alt

Selector universal i contextual

El selector universal es fa amb un asterisc * i serveix per agafar tots els elements de la pagina. Es pot fer servir sol o combinat amb altres.

Exemple bàsic: * { margin: 0; padding: 0; }

Tambe es pot fer servir com a contextual, combinat amb un altre selector per agafar tots els fills. Exemple:

.container-col-3 > * { width: 200px; float: left; border: thin solid black; }
En aquest cas, agafa tot el que esta directament dins de .container-col-3

Pseudoclasses per posició

Aquestes pseudoclasses serveixen per aplicar estils segons la posicio d’un element dins del pare.

  • :first-child – agafa el primer fill d’un element. Exemple: div:first-child { color: red; }
  • :last-child – agafa l’ultim fill. Exemple: div:last-child { font-weight: bold; }
  • :nth-child(n) – agafa el fill en la posicio que li diguis. Exemple: ul:nth-child(2) { font-size: 30px; }

Pseudoelements

Els pseudoelements serveixen per donar estil a parts concretes d’un element.

  • ::first-letter – dona estil a la primera lletra. Exemple: p::first-letter { font-size: 30px; }
  • ::first-line – dona estil a la primera linia del text. Exemple: p::first-line { font-weight: bold; }
  • ::before – dona estil abans de l’element. Exemple: h1::before { display: block; background-color: lightgray; }
  • ::after – dona estil despres de l’element. Exemple: h1::after { display: block; margin-top: 10px; }

Com agrupar selectors

Es poden posar varis selectors junts si vols que tinguin el mateix estil. Es separen amb comes.

Exemple: h1, h2, p { color: red; }

Amb aixo no cal repetir el mateix estil a cada etiqueta.

L’algorisme de CSS

Quan hi ha mes d’un estil que afecta al mateix element, el navegador ha de triar quin aplicar. Aixo es diu la «cascada».

1. Cascada – si dos estils afecten el mateix, el navegador escull el que esta mes abaix (l’ultim).

2. Especificitat – depen del tipus de selector:

  • Etiqueta p nivell 1
  • Classe .text nivell 2
  • ID #titol nivell 3

Si hi ha conflicte, guanya el que te mes especificitat.

3. Herència – alguns estils es passen dels pares als fills. Per exemple, el color del text del body afecta als p si no tenen color propi.

El model de caixa

El model de caixa es com el navegador dibuixa els elements HTML. Cada element te un espai interior i exterior.

  • Content – el contingut com text o imatges
  • Padding – espai entre el contingut i el border
  • Border – el contorn de l’element
  • Margin – espai de fora que separa de la resta

Aixo es pot controlar amb la propietat box-sizing. Hi ha dos valors:

  • content-box – es el valor per defecte. El width nomes aplica al contingut, el padding, el border i el margin es sumen a fora.
  • border-box – el width inclou el padding i el border. Tot queda dins del tamany fixat. Menys el margin que segueix a fora

Exemple:
*, *:after, *:before { box-sizing: border-box; }

Caixa de linia i de bloc

Hi ha elements que son de bloc i altres que son de linia.

  • Bloc – ocupa tota la linia i baixa avall. Exemple: div, p, h1
  • Línia – es posa al costat d’altres i no fa salt. Exemple: span, a, img

Es pot canviar amb display. Exemple: span { display: block; } o div { display: inline; }

Aixo serveix per canviar com es veu l’element a la pagina.

El col·lapse dels marges

Quan poses dos marges seguits a vegades es solapen i nomes conta el mes gran. Aixo es diu col·lapse.

Col·lapse vertical:

Passa quan dos elements estan un sota l’altre, com div seguits. Si un te margin-bottom: 30px i l’altre margin-top: 20px, nomes s’aplica el de 30px.

Col·lapse horitzontal:

Aixo no passa igual. Quan els marges estan al costat (per exemple, elements en linia), els marges si que se sumen.

Es pot evitar el col·lapse amb padding o border.

El posicionament

El position serveix per moure elements a la pagina. Pero segons el valor que poses, es comporta diferent.

  • static – es el valor que tenen tots per defecte. No es pot moure amb top, left, etc.
    Exemple: div { position: static; }
  • relative – es mou des d’on estava. Es com si estigués en el mateix lloc, pero pots moure’l una mica.
    Exemple: div { position: relative; top: 20px; } → baixa 20px
  • absolute – es mou respecte al seu pare si el pare te position. Si no, es mou respecte al body.
    Exemple: div { position: absolute; top: 50px; left: 100px; }
  • fixed – queda enganxat a la pantalla, encara que facis scroll. Serveix per fer menús que no es mouen.
    Exemple: div { position: fixed; bottom: 0; right: 0; }
  • sticky – es mou amb el scroll, pero quan arriba a un punt es queda clavat allà. Es mig fixe mig normal.
    Exemple: div { position: sticky; top: 0; }

La diferència entre relative i absolute es aquesta: amb relative l’element es manté al seu lloc, ocupa el seu espai normal, pero el pots moure una mica amb top, left, etc. Com si l’estiguessis desplaçant una mica. En canvi, amb absolute, l’element surt del lloc on estava i ja no ocupa espai allà. Va flotant i es col·loca on tu li diguis. Si no li dius res, se’n va cap a l’esquerra a dalt de tot. Si li poses absolute, es posiciona respecte al primer element pare que tingui position. Si cap pare te position, llavors es mou respecte al body.
