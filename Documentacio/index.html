HTML

Estructura basica HTML

  • <head>
  • <body>
  • <meta>
  • <title>
  • <link>
  • <script>

<head>: conte les metadades del document, com el title que surt a la pestanya, links a fulls d’estil i meta per indicar codificacio i viewport.

<body>: hi va tot el contingut visible de la pagina: textos, imatges, enllaços, formularis, etc. Mejor no carregar-lo massa per no alentir el carregament.

<meta>: dona info que no es veu. Els mes normals son:

  • charset="UTF-8" – defineix la codificacio de caracters.
  • name="viewport" content="width=device-width, initial-scale=1.0" – ajusta amplada i zoom per movils.

<title>: posa el titol del document, que apareix al navegador i tambe als resultats de cerca.

<link>: enllaça amb recursos externs, normalment CSS. Per exemple:
<link rel="stylesheet" href="assets/css/bootstrap.css" />

<script>: inclou o vincula codi JavaScript. Es sol posar al final del <body> per no bloquejar el render:

<script src="assets/js/bootstrap.bundle.js"></script>

Etiquetes estructurals i web semantica

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>

Va servir per organitzar la pagina amb tags mes clars. Jo crec que es mes facil de llegir.

<header>: va a l’inici, amb el titol o logo i menu.

<nav>: links de navegacio, com menu principal.

<main>: el contingut principal, només un per pagina.

<section>: parts separades de tema dins el main.

<article>: coses independents, com noticies o posts.

<aside>: info extra al costat, coses complementaries.

<footer>: al final, amb info de contacte o drets.

Etiquetes de text

  • <h1>-<h6>
  • <p>
  • <span>
  • <b>
  • <i>

<h1>-<h6>: titols de diferents nivells per organitzar jerarquia de contingut.

<p>: paragraf de text, es fa servir per blocs de contingut.

<span>: es un text en linia, util per aplicar estils.

<b>: posa text en negreta, sol per destacar.

<i>: posa text en italica.

Imatges

  • <img>
  • <picture>

Es fan servir per afegir imatges amb opcions de formats segons pantalla.

<img>: etiqueta basica per posar una imatge amb src i alt.

<picture>: envolta variants amb <source> per triar format o resolucio segons dispositiu.

Enllaços

Per fer un enllaç s’utilitza l’etiqueta <a>. Dins hi poses el text que vols que sigui clicable, i amb href li dius on ha d’anar.

Si poses un href amb una web com https://..., va a una pagina web. Si poses # amb un ID, va a una part de la mateixa pagina html.

Si vols que s’obri en una pestanya nova, has d’afegir target="_blank", sino recarrega la pagina i es veu la web clicada.

Exemple: <a href="https://www.youtube.com/" target="_blank">Youtube</a>

Llistes

Hi ha dos tipus principals: no ordenades i ordenades.

  • <ul>: llista no ordenada, amb punts.
  • <ol>: llista ordenada, amb numeros.
  • <li>: cada element de la llista, va dins de <ul> o <ol>.

Exemple llista no ordenada:

<ul>
  <li>Pa</li>
  <li>Llet</li>
  <li>Aigua</li>
</ul>

Exemple llista ordenada:

<ol>
  <li>Encendre l’ordinador</li>
  <li>Obrir el Google</li>
  <li>Entrar a la web</li>
</ol>

Taules

Una taula serveix per posar dades com si fos una graella. Té files i columnes.

Es fa servir <table> per crear-la. A dins s’hi posen les files amb <tr>. Després, a cada fila, es posen les cel·les: <td> per dades normals, i <th> si es vol cabeçera.

Exemple de taula:

Animal Habitat Alimentacio
Lleo Savana Carnivor
Os polar Artic Carnivor
Elefant Selva Hervivor

Audio i Video

Video
  • <video> amb controls
  • <source> amb ruta i tipus
Audio
  • <audio> amb controls
  • <source> amb mp3 o ogg

<video>: serveix per posar videos que es poden veure a la pagina. Si poses controls, surt el play, pausa i tot aixo. Dins s’hi posa un <source> amb la ruta i el tipus.

<audio>: paregut al video pero nomes per so. Tambe porta controls i dins un <source> amb l’arxiu mp3 o ogg.

Iframe

L’etiqueta <iframe> serveix per mostrar una altra web o contingut dins la teva pagina.

Es pot utilitzar per posar videos de YouTube, mapes de Google o una altra web sencera.
